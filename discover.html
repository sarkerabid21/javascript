Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML allows to insert or retrieve the HTML content inside an element.innerText returns only the visible text content of an element. textContent returns all text content inside an element, includinghidden text.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event happens on a child element, it first runs on that element and then moves up to its parent and ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement(tagName): Creates a new HTML element. appendChild(element): Adds an element as the last child of another element. removeChild(element): Removes a child element from a parent element.
